# eseed window
`namespace esd::window`

## Goal
A cross-platform, modern window library for C++ supporting a variety of rendering apis (such as Vulkan, OpenGL, etc.)

## Features

### Planned
- Platform-agnostic construction
- Window information
  - Sizing
  - Positioning
  - Fullscreen
  - Title
  - Icon
- Input and event handling
  - Cursor position
    - Window and screen
  - Keys
- *More to come*
